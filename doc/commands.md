 
Commands in terminal
====================


List directory content
----------------------

`ls` - list file

Change directory
----------------

cd - change directory 

### examples


```
cd nordadventure
cd ..
```
  
  ``
Make directory
----------------

mkdir - make directory

### examples
```
mkdir temp
```

Open file
---------

open - open file 
### example 
```
open radio.html
```


Open file or directory in Sublime text editor
----------------------------------------------

subl <filename> - start sublime editor 
subl . - open folder for editing in sublime
